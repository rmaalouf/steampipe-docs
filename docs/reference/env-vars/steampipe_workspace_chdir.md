# STEAMPIPE_WORKSPACE_CHDIR
Sets the Steampipe workspace directory. If not specified, the workspace directory will be set to the current working directory.

## Usage 
Set your api token:
```bash
export STEAMPIPE_WORKSPACE_CHDIR=~/my-steampipe-mods
```
